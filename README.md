# Holi-Celebration
Holi Wishes
Happy Holi - Shivani Karna

A beautiful, interactive Holi celebration website with colorful confetti, glowing text, and festive sounds!

Features
- Glowing, color-changing name animation
- Interactive color-burst button
- Rainbow confetti and emoji particles
- Web Audio API background music (piano tune)
- Piano chime sound effect on click
- Animated gradient background

How to Set Up and Deploy

Step 1: Create Folder Structure
1. Create a new folder on your computer named holi-celebration
2. Inside it, create 4 files:
   - index.html
   - styles.css
   - script.js
   - README.md
3. Copy the content of each file into the respective files

Step 2: Upload to Google Drive
1. Go to Google Drive: https://drive.google.com
2. Create a new folder called holi-celebration
3. Upload all 4 files to this folder
4. Right-click the folder → Share → Anyone with the link

Step 3: Deploy to Netlify (Drag and Drop)
1. Download the holi-celebration folder from your computer
2. Go to Netlify: https://app.netlify.com
3. Sign in (free account with Google or GitHub)
4. Click "Add new site" → "Deploy manually"
5. Drag and drop the holi-celebration folder
6. Your site is live! Copy the URL and share!

Customization
- Edit name in index.html: change "Shivani Karna"
- Edit message in index.html: change wishes
- Change colors in script.js holiColors array
- Adjust animation speeds in styles.css

Audio
- No external audio files needed!
- Uses Web Audio API for sound generation
- Piano chime plays on button click
- Background music plays on first click
- Works offline

Browser Support
- Chrome, Firefox, Safari, Edge (all modern browsers)
- Mobile responsive
- Works offline
